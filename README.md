## Guide to boot macOS
### Warning: This build is just compatible with Lenovo Ideapad Gaming 3. With another laptop, go Dortania guide to build from scratch (just follow the guide, don't watch youtube)
1. Adding the EFI folder + com.apple.recorvery.boot (version you need) to an USB
2. Plugging the USB to laptop, open bios menu, let USB boot first and save
3. Choose EFI to boot. In the recorvery mode, choose partition to download MacOS, changing it to AFPS type and save
4. (Boot without USB) After go into MacOS, use disk utility to shrink 200mb partition, change it to FAT type, move EFI folder from USB to that partition, go to BIOS mode and let this partition first and done <3
